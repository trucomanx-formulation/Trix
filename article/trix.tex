\section{Trix}
\subsection{Definition}

A trix $\mathbf{A}\in \mathbb{R}^{M\times N\times L}$ is an array in $3D$, with elements $a_{mnl}$, 
$\{m,n,l\}\in\mathbb{N}$, $1 \leq m \leq M$, $1 \leq n \leq N$, $1 \leq l \leq L$.
